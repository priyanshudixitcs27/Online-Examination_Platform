# 🚀 PBL Nexus: Modern Project Management Workspace

**PBL Nexus** is a premium, purpose-built Project-Based Learning (PBL) management system designed for academic environments. It bridges the gap between students and faculty by providing a centralized workspace for milestone tracking, task management, and digital grading.

---

## 📸 Project Interface
* **Student Portal:** Featuring a Kanban task board, progress tracking, and file submission.
* **Faculty Admin:** Featuring class-wide performance analytics, student review queues, and grading logic.

---

## 🛠️ Tech Stack
This project is built using a modern frontend architecture to ensure high performance and a professional UI/UX:

* **HTML5:** Semantic structure for a Single Page Application (SPA) experience.
* **Tailwind CSS:** Utility-first styling for a high-fidelity aesthetic.
* **JavaScript :** Dynamic DOM manipulation for page switching and role-based logic.
* **Google Fonts:** Utilizing 'Plus Jakarta Sans' for premium typography.

---

## ✨ Key Features

### 👨‍🎓 For Students
- **Real-time Progress:** Visual progress bars for project milestones.
- **Kanban Task Board:** Manage tasks through "In Progress" and "Completed" states.
- **Direct Submission:** Simplified file upload interface for project deliverables.

### 👩‍🏫 For Faculty (Admin)
- **Review Queue:** A dedicated list of all student submissions ready for grading.
- **Class Analytics:** Visual bar charts representing class-wide performance and submission rates.
- **Role-Based Access:** Secure toggle between student and teacher views.
